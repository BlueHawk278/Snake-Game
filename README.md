# 🎮 Snake

## 🧩 Overview

**Snake** is a Python-based game built using Turtle Graphics
Your mission is to move the snake and collect the turtles

## 🚀 Features
- 💾 Save / Load system
- WASD and Arrow Key support

## 🕹️ Controls

| Action | Key / Control |
|--------|----------------|
| Move Up | `W` / `↑` |
| Move Down | `S` / `↓` |
| Move Left | `A` / `←` |
| Move Right | `D` / `→` |
